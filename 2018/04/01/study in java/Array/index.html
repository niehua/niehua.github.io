关于int数组的那些事

  1. int[] var=new int[10];申请10个int空间，默认为0
  2. int[] var=new int[2]{1,2};错误；int[] var=new int[]{1,2}；正确。不能加下标
  3. int[] var={1,2};T
  4. int[] var;var=new int[]{1,2};T

字节的负数编码

  1. 第8位为1表示负数，所以正数默认7位
  2. 为负数时，虚拟机自动添加高24位补1，共32位。eg:-99=11111111 11111111 11111111 10011101
  3. 负数时，取反加1；eg:-99=1001 1101(0110 0010+1=0110 0011)
  4. 正数时，取反加1；eg:-99=-110 0011(1 001 1100+1=1001 1101)
public class T
{
    public static void main(String[] agrs) 
    {
        byte[] b =new byte[]{80,-99};
        { 
            int i=b.length;
            while(i-->0)
            {
                String binary=Integer.toBinaryString(b[i]);//二进制
                String hex=Integer.toHexString(b[i]);//16进制
                System.out.println(b[i]);//10进制
                System.out.println(binary);
                System.out.println(hex);
            }    
        }
    }
}
