重叠的矩形

Posted on 2018-04-04 | In Java | Visitors:
2018年网易本科实习生上机考试题

题目

平面内有n个矩形, 第i个矩形的左下角坐标为(x1[i], y1[i]), 右上角坐标为(x2[i], y2[i])。

如果两个或者多个矩形有公共区域则认为它们是相互重叠的(不考虑边界和角落)。

请你计算出平面内重叠矩形数量最多的地方,有多少个矩形相互重叠。

方法一

/*

  • 参考大神的解题思路，https://www.nowcoder.com/discuss/70736?type=0&order=0&pos=15&page=1
  • 直接暴力求解，针对每个点在所有矩形中的次数，返回最大值
  • 分别枚举所有的横纵坐标,挨着判断每个矩形是否符合条件，计数维护最大即可。
  • 时间复杂度
  • O(n^3)
  • */
import java.util.*;
public class T 
{
    public static void main(String[] args)
    {
        Scanner sc = new Scanner(System.in);
        while (sc.hasNext())
        {
            int n = sc.nextInt();

            int[] x1=new int[n];
            int[] y1=new int[n];
            int[] x2=new int[n];
            int[] y2=new int[n];
            int i;
            List<Integer> xl=new ArrayList<Integer>();
            List<Integer> yl=new ArrayList<Integer>();
            for(i = 0; i < n; i++){//左下角横坐标
                x1[i]=sc.nextInt();
                xl.add(x1[i]);
            }
            for(i = 0; i < n; i++){//左下角纵坐标
                y1[i]=sc.nextInt();
                yl.add(y1[i]);
            }
            for(i = 0; i < n; i++){//右上角横坐标
                x2[i]=sc.nextInt();
                xl.add(x2[i]);
            }
            for(i = 0; i < n; i++){//右上角纵坐标
                y2[i]=sc.nextInt();
                yl.add(y2[i]);
            }
            int result=0;
            int temp=0;
            for(Integer x:xl)
            {
                for(Integer y:yl)
                {
                    temp=0;
                    for(i=0;i<n;i++)
                    {
                        //强制要求矩形的左下角坐标在另一个矩形左下角和右上角中
                        if(x1[i]<=x && y1[i]<=y && x2[i]>x && y2[i]>y) 
                        {
                            temp++;
                        }
                    }
                    result=Math.max(result,temp);
                }
            }
            System.out.println(result);
        }

    }
}
