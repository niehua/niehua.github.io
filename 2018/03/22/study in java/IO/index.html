知识点

  1. long :System.currentTimeMillis();单位毫秒
  2. System最终类的静态属性 public static InputStream in;
    当做为输入流时（System.in）返回的必然是InputStream“子类的对象”
    且实现了它的抽象方法的，因为InputStream是抽象类不能被实例化的，运用了多态的概念
  3. Java本身是不能往系统文件里面写入数据的，需要调用系统底层读写，体现在刷新流动作上。

  • 打印文本文件Demo

    import java.io.*;
    public class T
    {

    public static void main(String[] args) throws IOException
{ 
    FileReader fr=new FileReader("T.java");//当前目录要输出的文件名
    char[] buf=new char[1024];//一般给2k空间
    int num=0;
    while((num=fr.read(buf))!=-1)
    {
        System.out.print(new String(buf,0,num));
    }
    fr.close();
}

    }

完成一个简单文本文件复制demo

import java.io.*;
public class T
{
    public static void main(String[] args) 
    {  
        FileReader fr=null;
        FileWriter fw=null;
        try
        {
            fr=new FileReader("demo.txt");//读
            fw=new FileWriter("demo_copy.java");//写
            char[] buf=new char[1024];//是以字符格式读入，不是字节格式
            int length=0;
            while ((length=fr.read(buf))!=-1)
            {
                fw.write(buf,0,length);
            }
        }
        catch (IOException e)
        {
            throw new RuntimeException("读写错误");
        }
        finally
        {
            if(fr!=null)//避免空指针异常
            {
                try
                {
                    fr.close();
                }
                catch (IOException e)
                {
                    System.out.println("读文件关闭错误");
                }

            }
            if(fw!=null)
            {
                try
                {
                    fw.close();
                }
                catch (IOException e)
                {
                    System.out.println("写文件关闭错误");
                }

            }
        }

    }    
}

复制图片

/*
以字节流方式复制图片（用字符可能会乱码）

  1. 绑定文件
  2. 建立缓冲区

  3. 读取 写入
    */

    import java.io.*;
    public class T
    {

    public static void main(String[] args) 
{
    FileOutputStream fos=null;
    FileInputStream fis=null;
    try
    {
        fis=new FileInputStream("d:\\eg.jpg");//绝对路径文件名
        fos=new FileOutputStream("d:\\112.png");
        byte[] buf=new byte[1024];//1k缓存
        int len=0;
        while ((len=fis.read(buf))!=-1)
        {
            fos.write(buf,0,len);
        }
    }
    catch (Exception e)
    {
        System.out.println("读取异常");
    }
    finally
    {
        try
        {
            fis.close();    
        }
        catch (Exception e)
        {
            System.out.println("读取异常1");
        }

        try
        {
            fos.close();    
        }
        catch (Exception e)
        {
            System.out.println("读取异常2");
        }
    }

}

    }
