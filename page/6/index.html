Posted on 2018-03-20 | In Java | Visitors:

知识点

  1. 定义float类型数据结尾必须加。（定义时也必须）注意这里与c语言的区别，c定义的时候不用。
  2. byte范围[-128，127]
  3. 1 byte=8bit(位)
  4. 一个字符char两个字节，存一个汉字需要两个字节。存英文和汉字都占两个字节，但当用String存字母时，一个字母一个字节（减少存储空间），汉字不变。
  5. 与c/c++不同的：基本数据类型与boolean不能相互转换
  6. （静态）成员变量可不初始化，默认为0，但方法内的局部变量使用前必须初始化以及接口中的成员变量（默认静态）。
  7. Short a=128;正确，但是个对象。
  8. Double类最小值是正值2^(-1074);–1.0/0.0返回Infinity,在内存中，浮点数是无法等于0的，so被除数0.0是一个极小的浮点数

Integer a=new Integer(4);(a==4)true，
String a=new String(“cc”);(a==”cc”);//false
Long l=new Long(4);（l==a）报错
a.equals(l)；false
Integer b=new Integer(4);a.equals(b);true

面向对象与异常

Posted on 2018-03-19 | In Java | Visitors:

构造函数

  1. 子类构造函数默认调用父类空构造函数，所以若父类有含参构造函数，子类就必须显示声明调用父类哪一个构造函数或者父类定义一个空构造函数（原因看2）。
  2. 只要类中定义了构造函数（无论有无形参），JVM都不会提供默认构造函数。
    3.super只能表示父类的引用
  3. 每当对象创建后，JVM分配一个引用自身的指针-this，表示当前对象的引用，只能在类中的非静态方法中使用
  4. 重载的构造函数可以通过this()调用本类无参构造函数；若有继承。则在无参构造函数再调用父类构造函数。
